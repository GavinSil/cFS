## Header Selection

The correct header is `cfe_psp_sim_stepping.h` (main ESA header), NOT the shim header.

**Why main header required**:
- timebase_posix_clock.c calls CFE_PSP_SimStepping_Hook_GetTime()
- This function is declared in esa/fsw/inc/cfe_psp_sim_stepping.h
- The shim header only provides event reporting types (CFE_PSP_SimStepping_ShimEvent_t)

**Why include resolves**:
- esa_public_api INTERFACE exports both esa/public_inc AND esa/fsw/inc
- timebase_posix_clock links with esa_public_api when CFE_SIM_STEPPING=true
- Compiler finds cfe_psp_sim_stepping.h in esa/fsw/inc/ via transitive include directories
